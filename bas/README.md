# Розширення BAS Малий бізнес — панель «AI Агент»

Цей пакет вбудовує веб-панель AI-агента **всередину BAS Малий бізнес** окремим
пунктом лівого меню (як на референс-скріншоті). Логіка/аналітика лишаються в
нашому Python-сервісі; BAS лише показує панель через **Поле HTML документа** і
відкриває її у браузері.

> **Native-форму 1С не можна згенерувати з Python-проєкту** — її створюють у
> Конфігураторі BAS. Нижче — точні кроки + готовий код модулів (вставити copy-paste).

---

## 0. Передумови

1. Наш сервіс запущено і доступний з машини, де працює BAS:
   ```
   uvicorn main:app --host 0.0.0.0 --port 8000
   ```
   Перевірка з браузера тієї ж машини: `http://localhost:8000/dashboard` має відкритися.
   Якщо BAS на іншому ПК —替 `localhost` на IP сервера + відкрий порт 8000 у фаєрволі.

2. **Для вбудованого режиму (Поле HTML):** платформа 1С/BAS **8.3.21+** та встановлений
   **Microsoft Edge WebView2 Runtime** (на Windows 10/11 зазвичай вже є). Панель
   використовує `fetch`/`canvas` — старий IE-движок їх не підтримує. Якщо WebView2
   немає — використовуй варіант «кнопка → відкрити у браузері» (п. 6Б), він працює всюди.

3. Право вмикати розширення конфігурації (адміністратор).

---

## 1. Створити розширення

Конфігуратор → **Конфігурація → Розширення конфігурації** → кнопка «Додати»:
- Ім'я: `AIАгент`
- Префікс: `ai_`
- Призначення: `Доповнення`

Відкриється конфігурація розширення (порожня) — далі додаємо об'єкти в неї.

## 2. Константа з адресою панелі (щоб не правити код)

У розширенні: **Константи → Додати**
- Ім'я: `АдресПанелиAI`
- Тип: `Строка`, довжина `200`
- Значення за замовчуванням задамо у формі (нижче).

## 3. Підсистема (пункт у лівому меню)

**Підсистеми → Додати**
- Ім'я: `AIАгент`
- Синонім: `AI Агент`
- ✔ «Включати в командний інтерфейс»
- (за бажанням) встав картинку.

## 4. Обробка з формою-панеллю

**Обробки → Додати**
- Ім'я: `ПанельAIАгента`, синонім «Панель AI Агента»
- Вкладка **Форми → Додати** керовану форму `Форма` (зробити основною).
- Вкладка **Реквізити форми** (у редакторі форми, права панель «Реквізити»):
  додати реквізит форми `HTMLПоле`, тип **Строка**, ✔ необмежена довжина.
- На формі: перетягни елемент **Поле** → у властивостях `Вид` = **Поле HTML документа**,
  `ПутьКДанным` = `HTMLПоле`, розтягни на всю форму (Растягивать по гор/верт).
- Відкрий **Модуль форми** і встав код з `module_form.bsl`.
- (для кнопки-фолбеку) додай команду форми `ВідкритиУБраузері` + кнопку, код з `module_form.bsl`.

## 5. Прив'язати обробку до підсистеми

- У властивостях Обробки `ПанельAIАгента` → **Підсистеми** → ✔ `AIАгент`.
- Командний інтерфейс підсистеми: переконайся, що команда відкриття обробки
  (`ПанельAIАгента: форма`) видима. Тепер у лівому меню BAS з'явиться розділ
  «AI Агент», по кліку відкриється панель.

## 6. Оновити та запустити

- **Конфігурація → Оновити конфігурацію бази даних** (F7), прийняти зміни.
- Запустити BAS у режимі **1С:Підприємство** (F5).
- Зліва натиснути **AI Агент** → відкриється панель.

### 6А. Вбудований режим (Поле HTML) — основний
Працює, якщо платформа 8.3.21+ і є WebView2 (п. 0.2). Панель показується прямо у формі BAS.

### 6Б. Фолбек — кнопка «Відкрити у браузері»
Якщо HTML-поле порожнє/не рендерить (старий движок), використовуй кнопку на формі —
вона відкриває `http://<сервер>:8000/dashboard` у системному браузері. Працює на будь-якій платформі.

---

## 7. Що налаштувати в самій панелі (вже в браузері/HTML-полі)

Вкладка **Налаштування** панелі (наш Python):
- **Промпт агента** — системний промпт, зберігається у PostgreSQL `agent_config`, агент одразу його застосовує.
- **Інтервали розсилок** — година відправки, вікно повтору, поріг «мовчання», пауза, ліміт.
- **Кампанії** — увімк./вимк. reorder / win-back / новий товар.

Вкладка **Кампанії** — ручний запуск розсилки (кнопка «Запустити»), показує охоплення.

> Зміни застосовуються одразу: scheduler читає `agent_config` наживо.

---

## 8. Безпека (рекомендація)

Панель і API зараз без авторизації — це нормально для localhost/локальної мережі.
Якщо сервіс доступний ззовні — постав за реверс-проксі (nginx) з Basic Auth або
обмеж доступ по IP. Не виставляй порт 8000 у публічний інтернет без захисту.
